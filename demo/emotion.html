emotion

明道的表情插件,由于聊天对表情的特殊需求，原有表情插件无法满足，所以开发出这一套表情插件

设置表情的显示位置

在聊天时，我们需要将明道熊的表情单独作为一条消息发送，这时可以使用onMDBearSelect回调方法

要设置默认显示的表情面板，可使用defaultTab 默认为0，表示最近常用表情，1表示常见表情，2表示明道萌熊，3表示emoji表情

代码

            
                <div class="row">
                <button class="btn btnPrimary btnEmotion btnEmotionDefault" data-target="emotionDefault">
                <i class="icon-smile"></i>
                </button>
                </div>
                <textarea id="emotionDefault"></textarea>
            
        

            $(".btnEmotionDefault").emotion();

参数

名称 类型 默认值 描述
defaultTab Number 0 0表示最近常用表情，1表示常见表情，2表示明道萌熊，3表示emoji表情
placement String "top" 设置表情显示的位置，有四个参数："top","right","bottom","left"
mdBear Boolean false 设置是否开启显示明道萌熊表情
history Boolean true 是否显示最近使用的表情
imgPath String 表情文件的文件夹路径 当你希望使用自己的表情时，emotion插件将读取你配置的该目录下的表情文件

回调函数

onSelect(name,img) 在选中普通表情时触发，name:表情名称，img表情的图片src

onMDBearSelect(name,img) 在选中明道萌熊时触发,当你需要把明道萌熊单独作为一条消息时会很有用
