Model

弹出框Model是web页面中比较常见的一种交互形式

例子

直接jQuery调用，这是会自动在body下添加一个Modal的div


  $(".btn.btnNormal").mdBox(); //直接在要触发的元素上调用mdBox方法

你也可以自定义DOM层，然后使用参数target指定 {target:"#modal"}


  <div class="modal" id="model">
    <div class="modalContainer">
      <div class="box-header"> //头部
        <span class="btnClose" title="关闭"><i class="icon-closeelement"></i> </span>
      </div>
      <div class="box-content"> //内容
        <p class="h1">Hello,world!</p>
      </div>
    </div>
  </div>
        

  $(".btn.btnNormal").mdBox({target: "#model"}); //直接在要触发的元素上调用mdBox方法

除此之外，还可以使用data的方式直接直接调用，你不必再写任何的js代码，就能使用


  <button class="btn btnPrimary" data-toggle="model" data-target="#model">启动 Model</button>

你还能通过对象的方式调用，直接new MDModel()即可,注意！使用seaJS加载Modal模块时，在回调时应该返回MDModal对象，比如：


  seajs.use("mdModel/model", function (MDModel) {
    ...
  })
    

  <button class="btn btnObj">启动 Model</button>
        

  new MDModel({content: "new Obj"}).show();

很多时候你想凭空生成一个Model，这是你可以调用$.fn.mdModel.show()

参数设置

名称 类型 默认值 描述
target string null 目标Modal的选择器，当你希望自己创建Modal的DOM层时，需要指定
bgClose boolean true 点击背景遮罩是否关闭
content string null 设置要显示的内容，可以是文本，可以是dom标签。注意！一旦给他值则表示将有组件自动生成dom标签
onShow function null 回调函数，在box显示时调用
onClose function null 回调函数，在box关闭时调用
onDestroy function null 回调函数，在销毁Modal时触发

函数

destroy 移除Model并取消其触发事件

show 显示Model

hide 显示Model

$.fn.mdModel.show() 新增并显示一个Model

$.fn.mdModel.destroyAll() 销毁所有的Model
